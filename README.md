# GibbTV Website

Static website for [gibbtv.com](https://gibbtv.com) — an independent AI content network producing narrative-driven short-form video.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main landing page with channel cards |
| `sector47.html` | Sector 47 channel page |
| `terms.html` | Terms of Service |
| `privacy.html` | Privacy Policy |

## Design

- Black background, dark minimal aesthetic
- Georgia serif for headings, Arial sans-serif for UI text
- Gold accent (`#c8a96e`) for brand highlights
- Deep red palette for Sector 47 channel
- Mobile responsive at 600px breakpoint

## Deployment

Designed to be served as a static site. No build step required — open any `.html` file directly in a browser or deploy to any static host (Netlify, Vercel, GitHub Pages, etc.).
